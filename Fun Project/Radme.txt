# Dynamic Gravity Tic Tac Toe

## Description  
This project is a Python-based terminal game that enhances the classic Tic Tac Toe by introducing a dynamic gravity mechanic. Players can place their marks and also change the direction of gravity during the game, which rearranges all pieces on the board. The objective is to create a more strategic and interactive gameplay experience while practicing Python programming and game logic.

- Description of data:
