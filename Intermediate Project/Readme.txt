# 📊 Student Performance Analysis & Prediction

## Description  
The objective of this project is to analyze student performance data and build a machine learning model to predict students' average scores. The project helps in understanding how different factors like gender, parental education, lunch type, and test preparation affect student performance.

## Dataset  
- Source: Kaggle (StudentsPerformance Dataset)  
- Description of data:  
  The dataset contains records of students' academic performance with the following attributes:
  - Gender  
  - Race/Ethnicity  
  - Parental level of education  
  - Lunch type  
  - Test preparation course  
  - Math score  
  - Reading score  
  - Writing score  

## Steps Performed  
1. Data Cleaning  
   - Checked for missing values  
   - Verified data types  
   - Ensured data consistency  

2. Exploratory Data Analysis  
   - Used statistical summaries  
   - Identified patterns and relationships  

3. Visualization  
   - Created bar charts, histograms, and boxplots  
   - Visualized score distributions and comparisons  

4. Model Building (if applicable)  
   - Created a new feature: average score  
   - Applied Linear Regression model  
   - Split data into training and testing sets  

## Results  
- Key findings:  
  - Students who completed test preparation performed better  
  - Parental education has a noticeable impact on scores  
  - Lunch type also influences performance  

- Metrics (if applicable):  
  - Mean Absolute Error (MAE)  
  - Mean Squared Error (MSE)  

## Tools Used  
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn  

## Conclusion  
This project shows that data analysis and machine learning can effectively predict student performance. Various socio-economic and educational factors influence student scores, and predictive models can help identify trends and improve outcomes.
